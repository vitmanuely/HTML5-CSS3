Principais Formatações

Negrito / Destaque

Nessa Frase temos um termo em negrito usando a tag <b>
A tag <b> é uma tag não semântica, não possui significado.

Nesta outra frase temos um termo em destaque usando a tag <strong> e ela é semântica.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta outra frase agora possuimos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

E neste outro parágrado temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno utilizando respectivamente as tags <big> (Tag obsolenta) e <small>

Texto deletado

Podemos marcar um texto como exluido para indicar que ele deve ser lido, mas não considerado
utilizando a tag <del>

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.
utilizando a tag <ins>
Existe também o sublinhado com a tag <u> (Não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3
utilizando a tag <sup>

Texto Subrescrito

Para inserir coisas do tipo H2O
utilizando a tag <sub>
